# Request Example
It makes easy for me to code the project

## User API

Return the whole user.

```
GET user/:id
```

Create a new user.
```
POST user/
{"name": "The User Name",
"password": "secret",
... this is all you need, you can add as many keys as you like though
"other_key": "other_value",
"foo" : "bar"
}
```
